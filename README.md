# Projects Portfolio

[portfolio.mwid.dev](https://portfolio.mwid.dev/)

My developer project portfolio page. Made with [Pug](https://pugjs.org/) and [SASS](https://sass-lang.com/).

## Development

Install dependencies:

```bash
pnpm install
```

Build from source:

```bash
pnpm build
```

Serve static assets:

```bash
pnpm start
```
